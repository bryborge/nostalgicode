##Periodic Table of the Elements
A practice app written in AngularJS

###Developers
Bryan Borgeson

###Date
4/25/2015

###Purpose
To practice developing in AngularJS, while also developing a tool that is practical for students of chemistry.

###Description
This app allows a user to dynamically search for an element in the periodic table of elements list.  This app also allows the user to interactively bring up element information on wikipedia.

###Technologies Used
1. AngularJS / Angular UI Router
2. HTML5
3. CSS / Bootstrap 3

###Use and Editing
You can use and/or edit this application by:<br />
1. Cloning this project to the directory of your choice<br />
2. Opening the project in your favorite text-editor (optional)<br />
3. Open and point your browser to index.html<br />
*(ex: file:///project-directory/index.html)*
